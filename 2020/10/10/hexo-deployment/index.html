图床

因为域名备案的原因，不想再用七牛云的图床了，打算直接在 GitHub 上建一个仓库作为图床。

创建 qiweipeng/images 仓库。设置中打开 GitHub Pages，Custom domain 设置为 image.qiweipeng.com，打开 HTTPS。同时 GoDaddy 设置 CNAME，值 image 指向 qiweipeng.github.io。
