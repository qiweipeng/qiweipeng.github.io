数据结构学习笔记 - 08 映射

发表于 2020-12-06 更新于 2024-11-26

介绍

映射在有的语言也叫 字典，代表一种一一对应的关系。

主要的应用如：

  • 字典：单词–>释意
  • 名册：身份证号–>人
  • 车辆管理：车牌号–>车
  • 数据库：id–>信息
  • 单词统计：单词–>频率

映射也分有序映射和无序映射。有序映射基于搜索树实现，无序映射通常会使用哈希表实现。

集合和映射是很相似的，使用映射完全可以包装出一个集合的结构，只需不管 value 即可。
